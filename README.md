## 🚀 Overview
This project is an enterprise-level administrative ecosystem developed to automate workforce operations. It focuses on three core engineering challenges: **Geo-spatial validation**, **automated financial workflows**, and **high-performance rendering pipelines**.

## 🔬 Core Engineering Highlights

### 1. Geo-Fencing via Haversine Algorithm
Utilized the **Haversine formula** to enforce a "Digital Gate" for attendance. Access is granted only when the Euclidean distance between the user and the office perimeter is within the defined radius ($\pm 50m$ precision).
> *Ensures 100% location-verified attendance monitoring.*

### 2. High-Performance Rendering (160ms TBT)
Optimized the browser's critical rendering path using **CSS Layout Containment** (`contain: layout paint`). This isolates component reflows, achieving a **Lighthouse-verified Total Blocking Time of 160ms** and categorical scores of 100 in SEO, Accessibility, and Best Practices.

### 3. Zero-Trust Security Architecture
Engineered a stateless authentication layer using **JWT-signed HttpOnly cookies** to mitigate Session Hijacking (XSS/CSRF). Implemented rigorous Role-Based Access Control (RBAC) middleware for every administrative endpoint.
